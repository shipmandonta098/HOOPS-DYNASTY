# Commissioner Tools - Quick Reference Card

## 🚀 Quick Access

### Enable Commissioner Mode
Settings → Commissioner Mode → Enable (Red Section)

### Access Tools
1. **Click Badge** (top-right) → Panel with Add Player + Force Trade
2. **Player Page** → "👑 Commissioner Tools" dropdown → Edit/Delete/Injury
3. **Settings Tab** → Commissioner Mode section → All tools

---

## ➕ Add Player (One-Minute Guide)

**Access:** Commissioner Panel → "➕ Add Player"

**Required Fields:**
- First Name, Last Name
- Position (PG/SG/SF/PF/C)
- Age (18-45)

**Quick Defaults:**
- Height: 78" (6'6")
- Weight: 210 lbs
- OVR: 75, POT: 80
- Salary: $5M, 2 years

**Result:** Player created instantly ✅

---

## 🗑️ Delete Player (One-Minute Guide)

**Access:** Player Page → Commissioner Tools → "🗑️ Delete"

**Steps:**
1. Modal shows player info
2. ✅ Check confirmation box
3. Click "Delete Player"

**Result:** Player permanently removed ⚠️

---

## 🔄 Force Trade (One-Minute Guide)

**Access:** Commissioner Panel → "🔄 Force Trade"

**Steps:**
1. Select Team A
2. Select Team B
3. ✅ Check players to swap
4. Click "Execute Trade"

**Result:** Players instantly swap teams 🔄

---

## 🏥 Force Injury (One-Minute Guide)

**Access:** Player Page → Commissioner Tools → "🏥 Force Injury"

**Quick Select:**
- Minor = 5 games
- Moderate = 15 games
- Severe = 30 games
- Season = 82 games

**Result:** Player injured, unavailable for X games 🏥

---

## ⚠️ Quick Warnings

**Delete Player:** CANNOT BE UNDONE  
**Force Trade:** No salary cap checks  
**Force Injury:** Player unavailable during injury  
**All Actions:** Logged + News item created

---

## 🔒 Security

All tools require Commissioner Mode ON (red badge visible).

Disable: Badge → "🔒 Disable Commissioner Mode"

---

## 📊 What Gets Logged

Every action creates:
1. **Commissioner Log entry** (league.commissionerLog)
2. **News Feed item** (visible in News tab)

---

## 💾 Backup Before Major Changes

Settings → Data Management → "📥 Export League"
